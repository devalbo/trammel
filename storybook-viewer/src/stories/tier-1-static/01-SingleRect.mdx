# 001 — Single Rect

The simplest possible sprite: a single rectangle rendered with explicit absolute props.
No constraints, no references, no containers beyond the root `<Sprite>`.

## Elements

| Element    | Type      | Purpose            |
|------------|-----------|--------------------|
| `<Sprite>` | Container | Root SVG viewBox   |
| `<Rect>`   | Primitive | The rectangle      |

## Props Exercised

- `id` — Identity
- `x`, `y` — Absolute position
- `width`, `height` — Absolute dimensions
- `fill`, `stroke`, `strokeWidth` — Presentation
- `rx` — Corner radius
