# 005 — Single Path

A freeform SVG path using absolute coordinates. Tests passthrough of the `d` attribute.

## Elements

| Element | Type | Purpose |
|---------|------|---------|
| `<Sprite>` | Container | Root SVG viewBox |
| `<Path>` | Primitive | Freeform shape |

## Props Exercised

- `id`, `d`, `normalized` — Geometry
- `fill`, `stroke` — Presentation

## Default

<LiveCanvas code={`<Sprite viewBox="0 0 200 200">
  <Path
    id="arrow"
    normalized={false}
    d="M 40 100 L 140 100 L 140 70 L 180 110 L 140 150 L 140 120 L 40 120 Z"
    fill="#4a90d9"
    stroke="#2a5080"
    strokeWidth={1.5}
    rotation={45}
  />
</Sprite>`} />

## What This Validates

- Path passes the `d` string directly to SVG
- The `d` prop is not transformed or scaled
- Bounding box is computed from path data for anchor resolution (used by later examples)
