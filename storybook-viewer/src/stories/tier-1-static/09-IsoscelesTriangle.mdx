## Description

An isosceles triangle with two equal sides (legs). Can be defined by `base` + `height`, or `base` + `legs`. Depending on proportions, the triangle is acute (tall/narrow) or obtuse (short/wide).

## Elements

| Element | Type | Purpose |
|---------|------|---------|
| `<Triangle>` #tall | Primitive | Acute isosceles (height > base/2) |
| `<Triangle>` #wide | Primitive | Obtuse isosceles (height < base/2) |

## Syntax

```jsx
<Sprite viewBox="0 0 300 200">
  {/* Acute isosceles: tall and narrow */}
  <Triangle
    id="tall"
    kind="isosceles"
    base={40}
    height={80}
    x={40}
    y={20}
    fill="#4a90d9"
    stroke="#333"
  />

  {/* Obtuse isosceles: short and wide */}
  <Triangle
    id="wide"
    kind="isosceles"
    base={100}
    height={30}
    x={150}
    y={70}
    fill="#e74c3c"
    stroke="#333"
  />

  {/* Alternative: specify legs instead of height */}
  <Triangle
    id="legs_form"
    kind="isosceles"
    base={60}
    legs={50}
    x={40}
    y={120}
    fill="#f39c12"
    stroke="#333"
  />
</Sprite>
```

## Vertex Computation (for "tall", base=40, height=80)

- v0 (base-left): (40, 100) — bottom-left
- v1 (base-right): (80, 100) — bottom-right
- v2 (apex): (60, 20) — top-center
- Legs (v0→v2 and v1→v2): sqrt(20^2 + 80^2) ≈ 82.46

## Acute vs Obtuse

| Example | base | height | Apex angle | Classification |
|---------|------|--------|------------|---------------|
| `tall` | 40 | 80 | ~28 deg | Acute (all angles < 90) |
| `wide` | 100 | 30 | ~118 deg | Obtuse (apex angle > 90) |

The crossover point is when `height = base * sqrt(3)/2` (equilateral). Taller = more acute, shorter = more obtuse.

## What This Validates

- `kind="isosceles"` constructs with base centered and apex above
- Both `base + height` and `base + legs` forms work
- The same component produces both acute and obtuse triangles depending on proportions
