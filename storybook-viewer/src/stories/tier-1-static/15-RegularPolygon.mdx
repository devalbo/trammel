# Regular Polygon

The generic `Polygon` component renders any regular polygon via the `sides` prop. `Hexagon` and `Pentagon` are thin wrappers with `sides` pre-set.

## Elements

| Element | Type | Purpose |
|---------|------|---------|
| `<Polygon>` | Primitive | Regular n-gon |

## Props Exercised

- `sides` — Number of sides
- `sideLength` or `r` — Size (side length or circumradius)
- `centerX`, `centerY` — Center position
- `rotation` — Rotation around center

## What This Validates

- `sides` controls vertex count for any regular polygon
- Flat-bottom orientation is consistent across all side counts
- `sideLength` derives circumradius correctly for each n
- Rotation applies around the center point
