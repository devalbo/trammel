## Description

Four rects distributed to fill available space with equal gaps using `justifyContent="space-between"`.

## Elements

| Element | Type | Purpose |
|---------|------|---------|
| `<Layout>` | Container | space-between distribution |
| `<Rect>` x4 | Primitive | Distributed items |

## Syntax

```jsx
<Sprite viewBox="0 0 300 60">
  <Layout
    direction="row"
    justifyContent="space-between"
    alignItems="center"
    x={0}
    y={0}
    width={300}
    height={60}
    padding={10}
  >
    <Rect width={40} height={40} fill="#4a90d9" />
    <Rect width={40} height={40} fill="#2ecc71" />
    <Rect width={40} height={40} fill="#e74c3c" />
    <Rect width={40} height={40} fill="#f39c12" />
  </Layout>
</Sprite>
```

## Resolver Trace

1. Available width: 300 - 2×10 padding = 280
2. Total item width: 4 × 40 = 160
3. Remaining space: 280 - 160 = 120
4. Gaps (space-between, 3 gaps): 120 / 3 = 40
5. X positions: 10, 90, 170, 250
6. Y positions: (60-40)/2 = 10

## Expected SVG Output

```svg
<svg viewBox="0 0 300 60" xmlns="http://www.w3.org/2000/svg">
  <rect x="10" y="10" width="40" height="40" fill="#4a90d9" />
  <rect x="90" y="10" width="40" height="40" fill="#2ecc71" />
  <rect x="170" y="10" width="40" height="40" fill="#e74c3c" />
  <rect x="250" y="10" width="40" height="40" fill="#f39c12" />
</svg>
```

## What This Validates

- `justifyContent="space-between"` distributes remaining space as equal gaps
- First item touches left padding, last item touches right padding
- Combined with `alignItems="center"` for cross-axis centering
